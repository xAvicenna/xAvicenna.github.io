Building PCs!

I've had a interest for PCs for a few years, but haven't kept up to date much because I never really had the means to become experienced in building. That changed a few months ago when I decided that it was time to drop some cash and build myself a decent CP for gaming, media, and school.

Building was really fun dispite the issues I ran into along the way (namely a bad issue somewhere between the motherboard and CPU, but I'll discuss that alter)

My rig has an Fx8320e CPU which admittedly is the only thing I would change if I were to revise my build. For my GPU, I got a beautiful Asus Strix 970, and for my case I went with a NZXT S340! The case is one of the nicest budget cases there is (even though at the time it cost me $100 with shipping) and has amazing cable management.

Hardware list
  • Asus M5A99X EVO R2.0 ATX AM3+ Motherboard
  • AMD FX-8320E 3.2Ghz (OC'd to 4.1) CPU
  • Silverstone AR03 CPU cooler
  • Kingstone Fury 2x4GB RAM
  • Samsung 840 EVO 120GB SSD
  • Seagate Barracuda 2tb HDD
  • Asus STRIX 970 GPU
  • NZXT S340 case
  • Seasonic 620W PSU
  • Corsair AF140 QE case fans (x2)

Preferential and OS list

  • Windows 8.1 (might dualboot with Linux later)
  • Raser Naga laser mouse
  • KBC Poker ii keyboard (cherry mx blue)
  • Astro A40 headset + mixamp
  • Steelseries XL soft mousepad
  • Ikea Markus office chair
